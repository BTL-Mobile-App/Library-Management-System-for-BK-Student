# Library-Management-System-for-BK-Student

## Requirements
- React native
- NodeJS
- VSCode
- Android Studio hoặc thiết bị Android thật

## Installation
Clone the repository.
```bash
git clone https://github.com/BTL-Mobile-App/Library-Management-System-for-BK-Student.git
```

### Backend
```bash
cd backend
npm install
npm run dev
```

### Frontend (mobile)
```bash
cd frontend
npm install
```
Login credentials: `ndngocks@gmail.com` - `password1`

## Features
- Login/Logout
- Home screen: search bar, category list, book list
- View book detail
- Hold a book
- Hold ticket list: view, cancel
- Borrow ticket list: view, return QR, rate book
- Profile screen
