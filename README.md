# Library-Management-System-for-BK-Student

## Requirements
- React native
- NodeJS
- VSCode
- Android Studio hoặc thiết bị Android thật

## Installation
Clone the repository.
```bash
git clone https://github.com/BTL-Mobile-App/Library-Management-System-for-BK-Student.git
```

### Frontend (mobile)
```bash
cd frontend
npm install
```
Login credentials: `ndngocks@gmail.com` - `password1`

### Backend
```bash
cd backend
npm install
npm run dev
```

## Features
- Login/Logout
- Quick Search & Advanced Filters
- Book Details
- My Shelf (Borrowing Dashboard)
- Automatic Due Date Reminders (Push & Email)
- Hold / Reserve Queue
- Borrowing History by Semester
- QR / Barcode Lookup
- Profile & Privacy Settings
- Help / FAQ & Feedback
